# HarmsWay Hugo Theme

A custom Hugo theme built on PaperMod, featuring:

- Full-screen hero sections with background images
- Alternating trapezoid content blocks for visual interest
- Transparent header with blur effect
- Custom color system with CSS variables
- Responsive design for all devices
- Gallery layouts
- Portfolio-style homepage

## Installation

1. Add the theme to your Hugo site's `themes` directory
2. Set `theme = 'HarmsWay'` in your `config.toml`

## Configuration

See the example `config.toml` for customization options.
